# Title: Drone Method and Validation Code

Group Members: Alisha Manocha, Reagan Ross, Aydin Khan, Roberto Julian Campos, Kamran Hussian

## Project Summary:
The drone operates under a continuous feasibility constraint: it must always possess sufficient energy to decelerate to a stop and complete a return trip. A turn-around decision is triggered the moment forward progress would deplete the energy buffer required for a safe homecoming. For more details, see the write up.
